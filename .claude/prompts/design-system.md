# Design System Skill

프로젝트의 디자인 시스템을 분석하고 개선 방안을 제안합니다.

## 분석 항목

1. **Design Tokens 검토**
   - Colors, Typography, Spacing
   - Shadows, Breakpoints, Z-index

2. **Component 구조 분석**
   - Atomic Design 준수 여부
   - Component API 일관성
   - Composition patterns

3. **테마 시스템**
   - Dark mode 지원
   - 브랜드 테마 확장성

4. **문서화 수준**
   - 사용 가이드
   - 예시 코드
   - 접근성 노트

## 출력 형식

- 현재 상태 분석
- 개선 권장사항
- 우선순위별 액션 아이템
