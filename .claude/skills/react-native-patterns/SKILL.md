# React Native Patterns

## Purpose

This skill helps implement React Native applications with proper mobile patterns.

## Instructions

When advising on React Native:

1. **Structure Navigation**: React Navigation.
2. **Handle Styling**: StyleSheet and responsive.
3. **Integrate Native**: Native modules.
4. **Manage State**: Context or Redux.
5. **Optimize Performance**: FlatList and memoization.

## React Native Checklist

### Architecture
- [ ] Navigation configured
- [ ] Styling approach consistent
- [ ] Platform-specific files
- [ ] Native modules when needed
- [ ] State management chosen

### Performance
- [ ] FlatList for lists
- [ ] Images optimized
- [ ] Memoization applied
- [ ] Hermes enabled
- [ ] Bundle size managed
